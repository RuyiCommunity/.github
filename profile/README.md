# 如意 RISC-V 社区

如意 RISC-V 社区由中国科学院软件研究所牵头发起，联合产业伙伴及多家开源社区共同建设，致力于打造面向 RISC-V 的上游软件共建平台。

围绕基础软件支持与原生软件栈建设，社区持续推进内核、工具链、基础组件及关键基础设施建设，不断完善 RISC-V 软件生态基础能力，推动产业协同与生态繁荣。

---

## 社区项目

### 1. [openRuyi 如意原生操作系统](https://github.com/openruyi-project)
RISC-V 原生操作系统项目，打造 RISC-V OS 基准对照平台。

### 2. [RuyiSDK 如意开发工具集](https://github.com/ruyisdk)
RISC-V 原生开发工具集，面向开发者的全方位、集成式、全功能开发环境。

### 3. [RuyiAI 如意 AI 软件栈](https://github.com/RuyiAI-Stack)
RISC-V 原生 AI 软件栈项目，覆盖推理框架、算子库、AI 编译器等全栈 AI 关键软件。

### 4. [RuyiRepo 如意软件仓](https://github.com/ruyirepo)
RISC-V 开源制品仓库，实现 RISC-V 生态软件可靠供给。

### 5. [RVCK 如意内核同源工程](https://github.com/rvck-project)
RISC-V 内核同源工程，统一内核演进基线，应对生态碎片化风险。
